LeetCode#136.Single Number

发表于 2016-08-29

题目描述

Given an array of integers, every element appears twice except for one. Find that single one.

解法一

使用位异或操作求解。

解法二

使用HashSet求解，遍历数组，如果set中存在这个数，则将其从set中移出，若不存在，则在set中保存这个数。
